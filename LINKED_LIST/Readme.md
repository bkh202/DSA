# Linked List

This directory contains Java implementations of common Linked List algorithms and interview problems. The solutions are written with a focus on clean code, readability, and optimized approaches suitable for coding interviews.

Each problem includes:

* Problem description
* Approach explanation
* Algorithm
* Time & Space Complexity
* Key concepts
* Java implementation

---

## Topics Covered

* Linked List Basics
* Traversal
* Insertion
* Deletion
* Searching
* Finding the Middle Node
* Reversing a Linked List
* Merging Two Sorted Linked Lists
* Detecting the Intersection of Two Linked Lists
* Fast & Slow Pointer Technique
* Two Pointer Technique
* Dummy Node Technique
* Pointer Manipulation
* In-place Operations
