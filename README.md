# Linked List Data Structure in Java

A collection of **Linked List algorithms and interview problems** implemented in **Java**. This repository covers beginner to advanced concepts, including traversal, reversal, cycle detection, intersections, and other commonly asked coding interview questions.

---

## Repository Structure

```text
LINKED_LIST/
│
├── Cycle/
│   └── DetectCycle.java
│
├── Intersection/
│   └── IntersectionOfLinkedList.java
│
├── ReverseGroupK/
│   └── ReverseGroupOfK.java
│
└── ...
```

> More Linked List problems will be added over time.

## Problems Covered

| # | Problem                          | Algorithm                     | Time   | Space |
| - | -------------------------------- | ----------------------------- | ------ | ----- |
| 1 | Detect Cycle (HashMap)           | HashMap                       | O(n)   | O(n)  |
| 2 | Detect Cycle (Floyd's Algorithm) | Two Pointers                  | O(n)   | O(1)  |
| 3 | Intersection of Two Linked Lists | Two Pointers                  | O(m+n) | O(1)  |
| 4 | Reverse Nodes in K Groups        | In-place Pointer Manipulation | O(n)   | O(1)  |
